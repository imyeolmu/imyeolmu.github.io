### 파일 읽기, 쓰기
- 읽기 모드 : r
- 쓰기 모드 : w(파일이 존재하면 덮어쓰기, 없으면 새로 생성)
- 추가 모드 : a(파일이 없으면 새로 생성, 있으면 값을 추가)

- path 구분 : 상대경로('../' : 이전 폴더, './':현재 폴더)
  - 절대경로 : 'c:\user\abc'      


```python
fp = open("sample.txt", mode="w")
fp.write("hello python")
fp.close()
```


```python
rfp = open("sample.txt", mode="r")
content = rfp.read()
rfp.close()

print(content)
```
