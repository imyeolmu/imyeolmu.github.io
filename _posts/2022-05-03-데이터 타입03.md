### 데이터 타입

- 문자형
- char: 2byte 0 ~ 65535, 다국어 표현 가능
  

```java

package java_study;

	
public class JAVA0509_1{
	public static void main(String arg[]) {
		char ch= 'A'; // 홑따옴표 사용, 'A': ASCII코드값 65
		
		
		System.out.println("ch:"+ch);
	}
}	


```
```java
ch:A
```
