### 데이터 타입

```
- Primitive Type : 기본자료형
   
 - Reference Type : 참조형 ex) Demo d = new Demo();
   	.... 여기서 d는 참조형 변수
   	
   	: primitive type:
   		ㄱ) 수치형 
   			a) 정수형
   			b) 실수형
   		ㄴ) 문자형
   		ㄷ) 논리형 
   		
   	: Reference Type:
   		ㄱ) 클래스형
   		ㄴ) 인터페이스형
   		ㄷ) 배열
```

- 8진수(0~7)
```java

package java_study;


public class JAVA0509_1{
	
	public JAVA0509_1() {
		//앞에 0이 붙으면 8진수
		int a = 010; // 010 : 1*8^1 + 0*8^0 = 8
		int b = 011;// 011 : 1*8^1 + 1*8^0 = 9
		int c = 0341;// 3*8^2 + 4*8^1 + 1*8^0
		System.out.println("a:" +a);
		System.out.println("b:" +b);
		System.out.println("c:" +c);


	}
	public static void main(String arg[]) {
		new JAVA0509_1	();	
		
	}
}


```
```java
a:8
b:9
c:225

```
