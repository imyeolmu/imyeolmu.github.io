형변환을 해보자(casting)
---



- 정수에서 실수로 형변환 

```python
float(3) 
```

```
 3.0
```

&nbsp; 

- 실수에서 정수로 형변환

```python
int(3.6) 
```
```
3
```

&nbsp; 

- 정수에서 실수로 형변환


```python
str(100)
```
```
    '100'
```
&nbsp; 


strA을 형변환 해보기

```python
strA ="123" 
print(type(strA))

intA = int(strA)
print(type(int(strA)))


floatA = float(strA)
print(type(float(strA)))

# y = f(x)
# y = f( f(x) )

```
```
    <class 'str'>
    <class 'int'>
    <class 'float'>
```   

&nbsp; 

형변환을 이용해서 연산 결과를 출력해보기

```python

num1 =int (input()) #input은 문자열이므로 형변환
num2 =int (input())

print(num1 + num2)
print(num1 * num2)
```
```
    10
    20
    30
    200
```
