### 라이브러리 오류 문제

maven은 자동으로 빌드가 되면서 참조하고 있는 라이브러리나 다운로드 파일,링크가 깨지거나 lock의 경우가 발생한다 

깨지면 0KB ,1KB로 되는데 라이브러리가 깨지게 된다


- 해결방법


```
1.사용자계정/.m2/repository의 디렉토리로 간다
```
```
2.respository안의 모든것을 다 지운다 (안에꺼만 지우기 )
```
```
3.이클립스를 닫고 respository안의 폴더를 과감하게 지운다 
```
```
4.다시 이클립스를 실행한다 
```
실행하면 알아서 빌드됨
