### 연산자2 

1. 삼항 연산자
```java

package java_study;

	
public class JAVA0509_1{

public static void main(String[] args) {
    //삼항연산자

	int a= 50, b = 60;
	//a가 b보다 크면 a출력 아니면 60 출력
	int y = (a>b)?a:b;
	
	System.out.println(y);
	
}
 }	


```

```java
60

```
