### DI(Depndency Injectior)

DI
```
스프링 프레임 워크 코어 기능 

종속 객체를 생성 조립해주는 도구

부품 조립 
```
&nbsp;
- Dependency들을 조립하기

&nbsp;

1.composition has a :일체형 has a관계
```java
class A
{
    private B c;

    public A(){
        b = new B();
    }
}

```
자기가 가지는 관계가 있다 (붙박이)

B는 A의 부품이다 

B는  = Dependency(종속성,종속객체)


부품이라고 보는게 쉽다


&nbsp;


2.Assdciation has a

```java
class A
{
    private B b;
    public A(){
    }
    public void setB(B b){
        this.b =b;

    }
}
```
조립형으로 쓰인다 (하나하나 조립해서)

기업형은 조립형이 많이 사용된다




&nbsp;


- 붙박이형
```java
A a = new A();
```


- 조립형

```java

Bb = new B(); //Dependency
A a = new A();

a. setB(b); //Injection
```

부품을 조립을 바꿀 수 있는 장점이 있으나 부품을 조립해야한다 


&nbsp;

injection(조립-주입)할때 두가지의 방법이 있다 

1.Setter Injection

```java
Bb = new B(); //Dependency
A a = new A();

a. setB(b); //Injection
```

2.Contruction Injection(생성자)
```java
Bb = new B(); 
A a = new A(b);
```

&nbsp;


만약 조립하기 힘드면 서비스를 받으면 되는데 그 도구를 사용하면 된다 


설정을 해주면 스프링이 이 조립을 해준다
